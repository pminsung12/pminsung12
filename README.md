  2. **개인보다는 팀** : 외국에서 중학교를 다니며 얻은 가장 귀중한 경험은 새로운 사람들을 정말 많이 만난 것이었고 그때마다 깨달은 것은 세상에 참 다양한 사람들이 존재하는구나였습니다. 그렇기에 각자의 다름을 인정하고 상대방의 생각과 의견을 항상 존중할 수 있는 성격이 되었고 이러한 사고가 바탕이 되어 협업 프로젝트를 진행 할 때 항상 좋은 쪽으로 작용할 수 있다고 생각합니다.
  3. **목표설정 완료, 준비는 끝났다!** : 동기부여가 되기 전 저는 방향성을 잡지 못하고 시간을 많이 낭비했는데, 지난 1년 간 미래에 대한 고민을 정말 많이 해보면서 프로그래밍으로 진로를 정했고 Back-end 쪽을 파보자는 생각을 하면서 앞으로 나아가며 조금씩이지만 열심히 준비를 하고 있습니다. 앞으로를 더 기대해주시면 감사하겠습니다🙇
